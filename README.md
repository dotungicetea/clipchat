This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Evironment
```
Node version: 21.1.0
```


## Setup

#### Step 1
```
yarn
```


#### Step 2
Create ``.env`` file
```
cp .env.dev.example .envs
```


#### Step 3: Run development server
```js
yarn dev
```


#### Step 4

Access:
```
http://localhost:3000/
```

Enjoy and Finish !!!
